🏆 นโยบายความเป็นส่วนตัว

การพนันบาสเกตบอล • การเดิมพันฟุตบอล • เดิมพันกีฬา Chrome Extension

✅ ไม่เก็บข้อมูลส่วนตัว

1. ภาพรวม

ส่วนขยาย Chrome "การพนันบาสเกตบอล เดิมพันกีฬา" (ส่วนขยาย) ให้บริการแสดงข้อมูลราคาต่อรองกีฬาและทีเด็ดบ้านบอลเพื่อการอ้างอิงเท่านั้น นโยบายนี้ชี้แจงข้อมูลที่ส่วนขยายเก็บรวบรวม ใช้งาน และคุ้มครองอย่างชัดเจน

สรุปสั้น: ส่วนขยายนี้ ไม่เก็บ ไม่ส่ง และไม่แบ่งปันข้อมูลส่วนบุคคลใดๆ ทั้งสิ้น

2. ข้อมูลที่เก็บรวบรวม

ส่วนขยายเก็บเฉพาะข้อมูลการตั้งค่าที่จำเป็นต่อการทำงานในเครื่องของผู้ใช้เท่านั้น:

  • การตั้งค่าภาษา – บันทึกเฉพาะตัวเลือกภาษา (ไทย) ใน chrome.storage.local
  • ช่วงเวลาการรีเฟรช – บันทึกความถี่การอัปเดตราคาต่อรอง (ค่าเริ่มต้น: 5 นาที)

ข้อมูลดังกล่าวจัดเก็บไว้ในเบราว์เซอร์ของผู้ใช้เองโดยสมบูรณ์ ไม่มีการส่งออกไปยังเซิร์ฟเวอร์ภายนอกใดๆ

3. ข้อมูลที่ไม่มีการเก็บรวบรวม

ส่วนขยายนี้ ไม่ เก็บข้อมูลต่อไปนี้โดยเด็ดขาด:

  • ชื่อ นามสกุล หรือข้อมูลประจำตัว
  • ที่อยู่ IP หรือข้อมูลตำแหน่งที่ตั้ง
  • ประวัติการเข้าชมเว็บไซต์ (Browsing History)
  • คุกกี้หรือ Session ของเว็บไซต์ใดๆ
  • ข้อมูลการเงิน บัตรเครดิต หรือบัญชีธนาคาร
  • รหัสผ่านหรือข้อมูลการเข้าสู่ระบบ
  • ข้อมูลการพิมพ์ (Keystrokes) หรือ Clipboard

4. การใช้สิทธิ์ (Permissions)

ส่วนขยายขอสิทธิ์เฉพาะที่จำเป็นต่อการทำงานหลักเท่านั้น:

  • storage – บันทึกการตั้งค่าของผู้ใช้ (ภาษา, ช่วงเวลารีเฟรช) ไว้ในเครื่องผู้ใช้งานเท่านั้น ไม่มีการ sync ออกนอกเครื่อง
  • alarms – ตั้งตัวจับเวลาดึงข้อมูลราคาต่อรองกีฬา (การเดิมพันฟุตบอล / การพนันบาสเกตบอล) อัตโนมัติทุก 5 นาที เพื่อให้ข้อมูลเป็นปัจจุบัน

ไม่มีการขอสิทธิ์เข้าถึงแท็บ ประวัติ บุ๊กมาร์ก ดาวน์โหลด หรือข้อมูลระบบใดๆ

5. การแบ่งปันข้อมูลกับบุคคลที่สาม

ส่วนขยายนี้ ไม่แบ่งปัน ไม่ขาย และไม่โอน ข้อมูลใดๆ ให้กับบุคคลที่สาม บริษัทโฆษณา หรือบริการวิเคราะห์ข้อมูลใดๆ ทั้งสิ้น

6. ความปลอดภัยของข้อมูล

ข้อมูลการตั้งค่าทั้งหมดจัดเก็บผ่าน chrome.storage.local API ซึ่งได้รับการคุ้มครองโดย sandbox ของ Chrome โดยตรง ไม่มีการส่งข้อมูลผ่านเครือข่ายอินเทอร์เน็ต

7. เด็กและเยาวชน

⚠️ ส่วนขยายนี้เกี่ยวข้องกับข้อมูลการเดิมพันกีฬา ไม่เหมาะสำหรับผู้มีอายุต่ำกว่า 18 ปี และผู้พัฒนาไม่มีเจตนาเก็บข้อมูลจากผู้เยาว์โดยเด็ดขาด

8. การเปลี่ยนแปลงนโยบาย

หากมีการอัปเดตนโยบายความเป็นส่วนตัว จะมีการประกาศในหน้านี้พร้อมวันที่แก้ไขล่าสุด การใช้งานส่วนขยายต่อไปหลังจากการเปลี่ยนแปลงถือว่ายอมรับนโยบายใหม่

9. ติดต่อผู้พัฒนา

หากมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ สามารถส่งข้อความผ่าน Chrome Web Store Developer Dashboard หรือติดต่อผ่านหน้าสนับสนุนของส่วนขยาย
